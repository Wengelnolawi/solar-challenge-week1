# Solar Energy Analytics Challenge

This repository contains the Week 1 challenge for the 10 Academy program. The goal is to explore and analyze solar energy data for Benin, Sierra Leone, and Togo, and build an interactive dashboard to support strategic investment decisions.

## Project Structure
- `notebooks/`: Exploratory data analysis (EDA) per country.
- `data/`: Cleaned datasets (not tracked in Git).
- `app/`: Streamlit dashboard.
- `compare_countries.ipynb`: Comparison across countries.
